Detailed Description

Computes the intersections between two line segments in ISegmentStrings and adds them to each string.

The ISegmentIntersector is passed to a INoder.

The NodedSegmentString.AddIntersections(LineIntersector, int, int) method is called whenever the INoder detects that two SegmentStrings might intersect.

This class is an example of the Strategy pattern.

summary> A proper intersection is an intersection which is interior to at least two line segments. /summary>

Note that a proper intersection is not necessarily in the interior of the entire Geometry, since another edge may have an endpoint equal to the intersection, which according to SFS semantics can result in the point being on the Boundary of the Geometry.
