Detailed Description

Extends the BinaryReader class to allow reading values in the BigEndian format.

While BEBinaryReader extends BinaryReader adding methods for reading integer values (BEBinaryReader.ReadInt32) and double values (BEBinaryReader.ReadDouble) in the BigEndian format, this implementation overrides methods, such BinaryReader.ReadInt32 and BinaryReader.ReadDouble and more, for reading ByteOrder.BigEndian values in the BigEndian format.
