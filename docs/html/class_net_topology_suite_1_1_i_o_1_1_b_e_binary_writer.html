Detailed Description

Extends the BinaryWriter class to allow writing values in the BigEndian format.

While BEBinaryWriter extends BinaryWriter adding methods for writing integer values (BEBinaryWriter.Write(int)) and double values (BEBinaryWriter.Write(double)) in the BigEndian format, this implementation overrides methods, such BinaryWriter.Write(int) and BinaryWriter.Write(double) and more, for writing ByteOrder.BigEndian values in the BigEndian format.
