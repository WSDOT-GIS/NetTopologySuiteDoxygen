NetTopologySuite.IO.PostGis2GeometryHeader.PostGis2GeometryHeader ( BinaryReader  reader)

Macros for manipulating the 'flags' byte. A uint8_t used as follows: —RGBMZ Three unused bits, followed by ReadOnly, Geodetic, HasBBox, HasM and HasZ flags.
