Gets and Ordinates flag that indicate which ordinates can be handled.

Ordinates NetTopologySuite.IO.PostGis2Reader.HandleOrdinates
getset

Gets and sets Ordinates flag that indicate which ordinates shall be handled.

No matter which Ordinates flag you supply, Ordinates.XY are always processed, the rest is binary and 'ed with IGeometryIOSettings.AllowedOrdinates.
