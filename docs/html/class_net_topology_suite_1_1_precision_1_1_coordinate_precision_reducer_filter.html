Detailed Description

Reduces the precision of the Coordinates in a ICoordinateSequence to match the supplied IPrecisionModel. /summary>

Uses IPrecisionModel.MakePrecise(double). The input is modified in-place, so it should be cloned beforehand if the original should not be modified.
