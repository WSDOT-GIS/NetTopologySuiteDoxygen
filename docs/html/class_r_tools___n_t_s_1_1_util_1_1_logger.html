Detailed Description

This is a simple implementation of a Logger class. The purpose is to threshold output based on a verbosity setting, format messages similarly, and collect all message writes such that they can be redirected easily. You (and I) should probably use the some other library for this, but I haven't found one that suits my needs.

This doesn't implement much of the functionality possible with this interface. This could redirect messages to other text writers, forward messages to subscribers, etc.
